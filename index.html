Learn How to Speedrun Rainworld

This is a resource based site meant to help get people into the hobby of speed running the game of Rainworld's base game by informing the user of different things you'll need to look into if you wish to start speed running this game, such as the three slug cat options for base game, major glitches you'll need, movement techs, and any external resources

Meet The Slug Cats

Monk
This is the easy mode of slugcats having a nearly identical campaign to the survivor with the exception that mobs kind of ignore you.
Survivor
This is the regular difficulty of Rainworld, past this campaign it only gets more difficult.
Hunter
This campaign is the hard mode of Rainworld as in order to beat the game on this mode have to go to nearly every region.

Where to Start

In this section we'll be going over different techs and glitches needed to complete an any% speed run
Karma Caching

In Rainworld each region is divided by these gates that you need to pass through, however to pass through you need that specific karma value or a higher value (which increases by resting, decreases by dying). Resting however is time consuming so we get through it by using a glitch called karma caching which allows you to get to max karma by only resting once.

Guardian Skip

In order to beat Rainworld (usually the end goal of most categories for the game), you have to go into the depths and ascend, however usually that can't be done without going to the iterators (these giant robots that upgrade your karma), as there's guardians stopping you. However, actually quite a few glitches you can use to get past them.

Movement

In this section we'll be covering how to move in Rainworld, as there's actually so much more to it then just moving in a direction and jumping, as most of this aren't directly explained
Pole Movement

This is the only actual tech in this movement section, but usually climbing up poles can be pretty slow, so speed runners found a way to move up them faster. You can do so by hitting the jump button and forward button while in the animation repeatedly. What can take 5 seconds to climb easily gets cut in half, just by using this method.

Slide Pounce

Slide pouncing is probably the most used movement for speed running as you not only maintain a pretty consistent momentum, but it's also really efficient.  For this movement you have to be moving, then immediately crouch and hit the jump button (The slide part), then towards the end of the slide you hit jump again and it launches you forward and you can basically chain this movement with the roll.

The Roll

The roll is a crucial key factor in maintaining speed and to avoid fall damage in spots. A roll is initiated when you're falling from a height by holding down and a horizontal direction, but the most interesting part is that you can initiate a roll from the slide pounce when normally you really can't roll after a jump unless you also dropped down somewhere.
